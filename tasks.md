# PFAG — タスク一覧（Tasks）

本書は、`plan.md` に基づき PFAG を実装するための具体的なタスクを示す。  
各タスクは独立して実行可能であり、完了条件が明確であることを目的とする。

---

## 1. プロジェクトセットアップ

- [ ] リポジトリ構造を作成（src/, docs/, openapi/, tests/）
- [ ] Python 仮想環境を構築
- [ ] 依存ライブラリをインストール（PyMuPDF, OpenCV, FastAPI, pdfrw）
- [ ] `.gitignore` を整備
- [ ] ローカル実行環境（uvicorn）を準備

---

## 2. PDF → 画像変換（PyMuPDF）

- [ ] PDF を 300dpi で画像化する関数を実装
- [ ] マルチページ PDF のページループ処理を実装
- [ ] 画像の一時保存を TemporaryDirectory で行う
- [ ] 壊れた PDF の例外処理（422 エラー相当）を実装
- [ ] 単体テスト：複数ページ PDF が正しく画像化されることを確認

---

## 3. 下線検出（テキストフィールド）

- [ ] モルフォロジー演算による水平線強調処理を実装
- [ ] `min_line_width` パラメータを適用
- [ ] `line_gap_threshold` による途切れ線の連結処理を実装
- [ ] 検出線をバウンディングボックス化
- [ ] フィールド配置ルール（高さ・オフセット・インセット）を実装
- [ ] 単体テスト：明確な下線が正しく検出されることを確認

---

## 4. チェックボックス検出（矩形検出）

- [ ] `cv2.findContours` による輪郭抽出を実装
- [ ] 階層構造を用いた矩形候補の抽出
- [ ] アスペクト比フィルタ（0.8 < W/H < 1.2）を適用
- [ ] 面積閾値フィルタを適用
- [ ] 正方形化処理（短辺基準）を実装
- [ ] サイズ制約（8〜20pt）を適用
- [ ] 単体テスト：チェックボックスが正しく検出されることを確認

---

## 5. 座標変換（画像座標 → PDF座標）

- [ ] PDF サイズ（pt）と画像サイズ（px）を取得する処理を実装
- [ ] X 座標変換式を実装  
  `X_pdf = X_img * (Width_pdf / Width_img)`
- [ ] Y 座標変換式を実装  
  `Y_pdf = Height_pdf - (Y_img * (Height_pdf / Height_img))`
- [ ] アスペクト比差異の検証
- [ ] 単体テスト：座標変換の誤差が許容範囲内であることを確認

---

## 6. AcroForm フィールド生成

- [ ] `/Tx`（テキストフィールド）生成処理を実装
- [ ] `/Btn`（チェックボックス）生成処理を実装
- [ ] 既存 AcroForm がある場合の append モードを実装
- [ ] 命名規則 `field_p{page}_{type}{index}` を実装
- [ ] 生成 PDF が Acrobat で正常に入力可能であることを確認

## 7. PDF 出力処理

- [ ] 生成したフィールドを PDF に書き込み

- [ ] 出力 PDF をバイナリとして返却する処理を実装
